# 激进版双向MACD趋势策略

## 策略概述

这是一个基于MACD指标的双向（多空）趋势跟踪策略，专为BTC永续合约设计。策略采用15分钟执行周期，结合1小时和4小时趋势过滤，通过AI自动优化参数。

**策略类型**: 激进型
**交易标的**: BTCUSDT永续合约
**杠杆**: 16-18x
**执行周期**: 15分钟
**过滤周期**: 1小时 + 4小时

## 核心信号

### 做多信号（4种）

1. **Long Breakout（多头突破）**
   - 价格突破近期高点
   - 15分钟/1小时/4小时三周期趋势向上
   - MACD金叉，ADX显示趋势强度
   - 成交量放大确认

2. **Long Pullback（多头回调）**
   - 上升趋势中的回调买入
   - 价格回踩21EMA后反弹
   - 保持趋势结构完整
   - 适合震荡市中的趋势延续

### 做空信号（2种）

3. **Short Breakdown（空头跌破）**
   - 价格跌破近期低点
   - 三周期趋势向下
   - MACD死叉，ADX确认
   - 成交量放大

4. **Short Bounce-Fail（空头反弹失败）**
   - 下降趋势中的反弹做空
   - 价格反弹至21EMA后回落
   - 趋势延续信号

## 技术指标体系

### 趋势判断
- **EMA系统**: 快线20/慢线55（15分钟）
- **MACD**: 12/26/9标准参数
- **ADX**: 趋势强度过滤（8-20阈值）
- **Choppiness**: 震荡过滤（<60-70）

### 入场过滤
- **多周期确认**: 15分钟+1小时+4小时同向
- **成交量**: 相对20周期平均（0.3-1.0倍）
- **RSI**: 防止极端超买超卖
- **ATR**: 波动率确认

### 退出机制

1. **分批止盈**
   - 首次止盈: 18-28%（平仓6-20%）
   - 追踪止盈: 激活后回撤12-18%触发

2. **保本止损**
   - 盈利17-23%后移动止损至保本

3. **ATR止损**
   - Breakout: 2.55倍ATR
   - Pullback: 1.85倍ATR
   - 做空信号: 1.3-1.7倍ATR

4. **趋势失效退出**
   - 小时线MACD反转
   - 价格跌破/突破小时快线

5. **时间止损**
   - Breakout: 最长180根K线（45小时）
   - Pullback: 最长108根K线（27小时）
   - 动态延长: 趋势强劲时可延长

## 风险管理

### 仓位管理
- **单仓大小**: 账户的20-23%
- **最小仓位**: 5000 USDT
- **最大仓位**: 30000 USDT
- **最大并发**: 5个仓位

### 加仓策略
- **触发条件**: 盈利12%且ADX>27
- **加仓大小**: 原仓位的35%
- **最多加仓**: 3次
- **加仓方向**: 仅Breakout/Breakdown信号

### 风险限制
- **最大回撤目标**: <30%
- **单笔最大亏损**: 42%（含杠杆）
- **爆仓容忍**: ≤6次/测试周期

## 数据集划分

采用Walk-Forward验证方法：

```
训练集: 2025-12, 2026-01, 2026-02（3个月）
验证集: 2026-03（1个月）
```

**设计理念**:
- 使用最近3个月数据训练（加密货币市场变化快）
- 用下个月验证泛化能力
- 每月滚动更新参数

## 评分函数

```python
综合分 = 训练集*0.30 + 验证集*0.70
        - 过拟合惩罚*1.2  # 训练>验证时扣分
        - 波动性惩罚*0.20  # 月度收益不稳定扣分
        - 回撤惩罚（>25%）*0.20
        - 爆仓惩罚*6.0
        - 手续费拖累*0.12
```

**验证集权重70%**: 强制策略在未见过的数据上表现良好

## Gate条件（准入门槛）

策略必须同时满足：
- 总交易 ≥ 40笔
- 训练集 ≥ 25笔
- 验证集 ≥ 10笔
- 最大回撤 ≤ 30%
- 爆仓次数 ≤ 6次
- 过拟合差距 ≤ 10分
- 验证集分数 > -5分

## 当前参数特点

### 入场条件（激进）
- **ADX要求极低**: 6.5-10（允许弱趋势）
- **Volume要求宽松**: 0.32-0.98（不强求放量）

- **追踪止盈激进**: 38-44%激活，回撤16-18%
- **止损较宽**: 2.35-2.55倍ATR

### 特殊设置
- **Fear&Greed**: 已禁用（不适合当前市场）
- **Pullback信号**: 极度宽松（volume只要0.32）
- **Breakout信号**: 宽松（volume只要0.46）

## 自动优化系统

### 优化流程
1. AI每5分钟生成新参数组合
2. 在训练集+验证集上回测
3. 计算综合分数
4. 通过Gate检查
5. 优于当前最优则保存

### 记忆系统
- 记录最近16次accepted/rejected参数
- 避免重复尝试相同方向
- 学习有效/无效的参数调整模式

### 通知机制
- Discord实时推送新最优策略
- 手机友好的垂直布局
- 包含完整性能指标
